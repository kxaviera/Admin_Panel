**Q: What if I only have 1 icon?**

**Q: Can I upload different styles?**
A: Yes, but it's better to use icons from the same pack for consistency.

**Q: What size should icons be?**
A: 512x512 px is ideal. Minimum 256x256 px.

**Q: Can I use JPG instead of PNG?**
A: Yes! But PNG with transparency looks better.

**Q: How do I remove a view?**

**Q: Will emoji still work?**
